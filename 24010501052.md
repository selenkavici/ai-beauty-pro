## 2. Projenin Amacı ve Kısa Açıklaması

AI Beauty Intelligence Platform, kullanıcıların saç modeli, saç rengi, cilt tonu ve güzellik tercihleri konusunda daha bilinçli kararlar verebilmesi için geliştirilmiş yapay zeka destekli bir web uygulamasıdır.

Kullanıcı sisteme yüzünün net göründüğü bir fotoğraf yükler. Uygulama, yüklenen fotoğraf üzerinden görsel tabanlı demo analiz gerçekleştirir ve kullanıcıya yüz şekli, cilt alt tonu, saç yoğunluğu ve stil enerjisi gibi sonuçlar sunar. Analiz sonucuna göre kişiselleştirilmiş saç modeli, saç rengi, makyaj/stil ve bakım önerileri oluşturulur.

Proje ayrıca AI chat asistanı, öneri detay modalı, randevu oluşturma sistemi, localStorage ile randevu saklama ve iletişim formu gibi özellikler içerir.

Bu proje final sunumu için geliştirilmiş olup dış API anahtarı gerektirmeden tarayıcı üzerinde çalışan demo analiz mantığıyla hazırlanmıştır.

## 6. Kurulum Adımları

Projeyi çalıştırmak için bilgisayarda **Node.js** kurulu olmalıdır.

GitHub reposunu klonlamak için:

```bash
git clone https://github.com/selenkavici/ai-beauty-pro.git
```

Proje klasörüne girmek için:

```bash
cd ai-beauty-pro
```

Gerekli paketleri yüklemek için:

```bash
npm install
```

---

## 7. Çalıştırma / Kullanım Talimatları

Geliştirme sunucusunu başlatmak için:

```bash
npm run dev
```

Komut çalıştıktan sonra terminalde verilen localhost bağlantısı tarayıcıda açılır. Örnek:

```text
http://localhost:5173
```

Production build almak için:

```bash
npm run build
```

Build sonucunda proje `dist/` klasörü içerisine derlenir.

## 8. Uygulama Kullanımı

1. Kullanıcı ana sayfada proje hakkında kısa bilgileri görür.
2. **AI Analiz** bölümünden yüzünün net göründüğü bir fotoğraf yükler.
3. Sistem yüz olmayan görsellerde kullanıcıya uyarı verir.
4. Yüz içeren fotoğraflarda demo analiz sonucu oluşturulur.
5. Analiz sonucunda yüz şekli, cilt alt tonu, saç yoğunluğu ve stil enerjisi gösterilir.
6. Kullanıcı **Öneriler** bölümünde kendisine özel güzellik önerilerini inceler.
7. Öneri kartlarında detay modalı açılabilir.
8. Kullanıcı AI chat asistanına saç rengi, saç modeli, cilt tonu veya randevu hakkında soru sorabilir.
9. Randevu formu üzerinden iş saatleri içerisinde danışmanlık randevusu oluşturabilir.
10. Oluşturulan randevular tarayıcı localStorage alanına kaydedilir ve sayfa yenilense bile görünür.
11. Kullanıcı isterse oluşturulan randevuyu silebilir.
12. İletişim formu üzerinden demo mesaj gönderimi yapılabilir.
